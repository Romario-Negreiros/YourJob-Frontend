<h1 align="center">YourJob</h1>

This is the backend for YourJob.com, built with nodejs, typescript and using express as framework.

### Libraries in use

- React router dom
- Lottie react
- Firebase
- Material UI
- Redux
- React-redux
- Redux toolkit
- React hook form
- React currency input field
